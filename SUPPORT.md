# Support

Thanks for using this repository. Here is where to go depending on what you need.

## Bug reports and feature requests

Please open an issue: https://github.com/neurot1cal/bdigital-public/issues

Use the bug or feature template, fill in the repro or motivation, and tick the
checklist items that apply.

## Questions and general discussion

Open a thread in GitHub Discussions:
https://github.com/neurot1cal/bdigital-public/discussions

Note: Discussions have to be enabled at the repository level. If the link above
returns a 404, discussions have not been enabled yet; open an issue labelled
`question` in the meantime.

## Tutorial-level help

The companion blog series walks through the samples in this repo at a
conceptual level, including design choices and how the pieces fit together.
Start there before filing an issue if your question is "how does this work?"
rather than "something is broken":

https://tech.bdigitalmedia.io/

## Security reports

Do not file security findings in public issues. See `SECURITY.md` for the
coordinated disclosure process.
